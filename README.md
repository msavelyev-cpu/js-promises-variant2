## Опис проєкту

Програма демонструє використання нативних Promise для роботи з файлами та обробки помилок.

Реалізовано:

* читання файлу;
* валідацію вмісту;
* трансформацію даних;
* збереження результату у файл;
* кастомні класи помилок;
* retry логіку для мережевих помилок;
* логування операцій;
* cleanup через finally.

## Встановлення та запуск

Перейти до папки проєкту:

cd js-promises-variant2

Запустити програму:

node app.js
